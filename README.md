# OmniAPK

Universal Android App Manager - Manage and update apps from multiple sources.

## Features

- Multi-source app discovery (Google Play, APKMirror, APKPure, F-Droid)
- App update checking
- Multiple installation methods (Standard, Root, Shizuku)
- Modern MVVM architecture with Hilt dependency injection
